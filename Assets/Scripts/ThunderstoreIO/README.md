Uses REPOLib to add a number of items as valuables to the game, mostly Star Trek themed. Some have custom sound effects. More valuables likely to be added in the future.

## Valuables
This mod currently adds:
- Antimatter Pod
- Sisko's Baseball
- Blue Barrel
- Saltahna Clock
- DaxMug
- Exocomp
- Betazoid Chime
- Horgahn
- Bottle of Kanar
- Kurlan Naiskos
- Metal Pipe
- DS9 Mug
- Odo's Bucket
- Pipe Lamp
- Pyramid Game
- Kirk's Strange Rock
- WideDoc Plushie
- Worf's Chair
- PicBust
